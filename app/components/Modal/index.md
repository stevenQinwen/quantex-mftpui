模态对话框

## 何时使用

需要用户处理事务，又不希望跳转页面以致打断工作流程时，使用 Modal 在当前页面中打开一个浮层，承载相应的操作

## API

### Extended Modal props

| 参数               | 说明               | 类型      | 默认值    |
| ---------------- | ---------------- | ------- | ------ |
| open         | 打开模态框            | Function(title, content, options)   | -     |
| close         | 关闭模态框             | Function()   | -     |



### 其他props请参考官网 [Modal](https://ant.design/components/select-cn/)
