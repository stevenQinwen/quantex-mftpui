输入框组组件。

## 何时使用

按配置展示输入框。

## API

### Extended TreeSelect props

| 参数               | 说明               | 类型      | 默认值    |
| ---------------- | ---------------- | ------- | ------ |
| items         | 输入框组配置项             | array   | -     |
| value         | 输入值             | array   | -     |
| disabled         | 禁用全部             | boolean   | -     |
