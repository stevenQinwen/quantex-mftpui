时间段选择组件。

## 何时使用

当用户需要选择一个时间段。

## API

### Extended TreeSelect props

| 参数               | 说明               | 类型      | 默认值    |
| ---------------- | ---------------- | ------- | ------ |
| value         | 已选项             | array   | []     |
| onChange        | 变化时回调函数 | function  | -    |
| valueFormat | 时间转换格式            | string  | -   |
