数字格式化组件。

## 何时使用

展示长数字时。

## API

### Extended TreeSelect props

| 参数               | 说明               | 类型      | 默认值    |
| ---------------- | ---------------- | ------- | ------ |
| value         | 输入值             | string or number   | -     |
| prefix         | 输入值的前缀             | string or element   | -     |
| suffix         | 输入值的后缀            | string or element   | -     |
| precision | 保留小数             | number   | 2     |
| nullValueContent | 空值时的显示内容            | string  | '--'   |
