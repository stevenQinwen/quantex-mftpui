自动转换时间戳组件。

## 何时使用

当用户需要选择一个时间段。

## API

### Extended TreeSelect props

| 参数               | 说明               | 类型      | 默认值    |
| ---------------- | ---------------- | ------- | ------ |
| value         | 已选项             | number or string   | -     |
| format | 时间转换格式            | string  | 'YYYY-MM-DD HH:mm:ss'   |
| originFormat | 原始数据格式   | string  | -   |
| nullValueContent | 空值时的显示内容   | string  | '--'   |
